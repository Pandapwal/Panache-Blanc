# panache-blanc

basic React SPA, Nodejs + express server api
built for a MongoDB database

===============

# start instances

run npm install in root and /frontend

## start mongoDb database
mongod

## start api server
cd /backend
node ./index.js 

## start client app
cd /frontend
npm start

## populate mongoDB with json files in
/backend/data-import

===============
